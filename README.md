Данный скрипт (компонент `Hud`) реализует простой **графический интерфейс (Heads-Up Display) для ПК-версии проекта**. Он предназначен для отображения прицела (crosshair) в центре экрана поверх основного содержимого, что особенно полезно в играх от первого лица или прицельных механиках.

**Основные возможности:**
- Загрузка изображения прицела через `AssetLink` – поддерживаются любые текстуры, доступные в проекте.
- Настройка размера прицела прямо в редакторе (параметр `crosshairSize`).
- Автоматическое центрирование спрайта при изменении разрешения экрана – скрипт отслеживает размеры GUI и пересоздаёт виджет, чтобы прицел всегда оставался по центру.
- Простота и минимализм – служит основой для дальнейшего расширения HUD (добавление здоровья, патронов и т.д.).

Компонент легко интегрируется в любой проект: достаточно прикрепить его к любой ноде, указать нужное изображение, и прицел появится на экране.
